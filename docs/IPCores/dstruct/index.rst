PoC.dstruct
===========

The namespace `PoC.dstruct` offers different data structure implementations.

**Package**

The package :ref:`NS/dstruct` holds all component declarations for this namespace.

**Entities**

 * :ref:`IP/dstruct_DoubleEndedQueue` implements a deque (two-sided FIFO).
 * :ref:`IP/dstruct_Stack` implements a regular stack.
